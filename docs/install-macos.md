# macOS 安装说明

本文档面向普通测试用户。

## 下载

从这里下载最新安装包：

- [GitHub Releases](https://github.com/fchangjun/AI-No-Token/releases)

推荐优先下载：

- `.dmg`

如果你更习惯手动解压，也可以使用：

- `.zip`

## 安装步骤

1. 下载最新 macOS 安装包
2. 打开 `.dmg`
3. 将 `AI-No-Token` 或对应应用拖到 `Applications`
4. 从 `Applications` 启动

## 如果系统提示无法验证开发者

当前版本没有 Apple Developer 签名，所以首次启动可能被 Gatekeeper 拦截。

如果出现拦截，请按下面任一方式处理。

### 方式 A：右键打开

1. 到 `Applications`
2. 找到应用
3. 右键点击应用
4. 选择“打开”
5. 在弹窗中再次选择“打开”

### 方式 B：系统设置里放行

1. 打开“系统设置”
2. 进入“隐私与安全性”
3. 在底部找到被拦截的应用提示
4. 点击“仍要打开”或对应放行按钮

## 首次使用

1. 启动桌面应用
2. 打开登录窗口
3. 登录你自己的 ChatGPT 账号
4. 捕获登录态
5. 使用本地 API

## API 地址

默认端口是：

```text
8787
```

你可以通过以下地址访问：

```text
http://127.0.0.1:8787
```

如果你和调用方在同一局域网，也可以通过当前电脑 IP 访问，例如：

```text
http://192.168.x.x:8787
```

## 常见问题

### 1. 应用提示损坏

当前发布版本不应再表现为“应用已损坏”。

如果仍然遇到：

1. 删除旧版本应用
2. 从最新 Release 重新下载安装包
3. 再按本文档中的“右键打开”方式启动

### 2. 启动后接口连不上

先检查桌面应用是否已经启动，然后访问：

```bash
curl http://127.0.0.1:8787/health
```

### 3. 聊天请求失败

网页方案可能受到：

- 登录失效
- 网页风控
- 页面结构变化

先重新登录，再重试。
